# Agent Run Checklist

## Before run
- [ ] Ho un `Task Card` valido
- [ ] Scope allineato a `agent-scope.policy.json`
- [ ] Definito `out of scope`
- [ ] Definiti comandi di validazione

## During run
- [ ] Modifico solo path consentiti
- [ ] Tengo traccia di decisioni e trade-off
- [ ] Apro escalation se incontro stop condition

## After run
- [ ] Build/Test/Lint eseguiti
- [ ] Handoff compilato (`templates/handoff.json`)
- [ ] Rischi residui documentati
- [ ] Nessuna modifica fuori confine
